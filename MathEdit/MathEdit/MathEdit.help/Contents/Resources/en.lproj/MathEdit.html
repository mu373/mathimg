About MathEdit

MathEdit is a project-based LaTeX equation editor for macOS. Keep all equations for a presentation or document in a single project, export them as SVG images, and re-import them anytime for editing. Every exported SVG contains embedded LaTeX source, so your equations are always editable.

Export equations as SVG or PNG. SVG files contain embedded metadata, allowing you to drag them back into MathEdit and continue editing.

Syntax

MathEdit documents use a plain-text format with LaTeX equations separated by ---. Use frontmatter to define color presets and \label{} to name equations.
