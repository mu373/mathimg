Document syntax

MathEdit uses a plain-text format where you write LaTeX equations separated by ---. Each equation can have a label and color settings.

Document structure

A MathEdit document consists of equations separated by three dashes:

E = mc^2
\label{energy}

---

F = ma
\label{newton}

---

a^2 + b^2 = c^2
\label{pythagoras}

Each section between --- separators becomes a separate equation that you can export individually.

Labeling equations

Use \label{name} to give each equation a descriptive name:

\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
\label{gaussian-integral}

Labels are used for:

  • Identifying equations in the sidebar
  • Naming exported files (e.g., gaussian-integral.svg)
  • Preserving equation identity when re-importing

If you don't provide a label, MathEdit assigns one automatically (eq1, eq2, etc.).

Frontmatter

Add a frontmatter section at the top of your document (before the first ---) to define settings:

define.myblue: #0066ff
define.myred: #ff0066
color: blue

---

E = \color{myblue} mc^2
\label{energy}

Frontmatter options:

define.name: value
Define a color preset that you can use with \color{name}
color: value
Set the default color for all equations

Per-equation colors

Override the default color for a specific equation by adding a comment at the end:

E = mc^2
% color: red

Or use \color{} within the equation to highlight specific terms:

E = \color{red} mc^2

Basic LaTeX syntax

Common LaTeX examples:

LaTeX Result
x^2 Superscript (x squared)
x_i Subscript
\frac{a}{b} Fraction
\sqrt{x} Square root
\sum_{i=1}^{n} Summation
\int_a^b Integral

Greek letters

Use backslash followed by the letter name:

LaTeX Letter
\alpha, \beta, \gamma α, β, γ
\Delta, \Sigma, \Omega Δ, Σ, Ω (uppercase)
\pi, \theta, \lambda π, θ, λ

Grouping with braces

Use curly braces {} to group multiple characters:

  • x^{10} — x to the power of 10
  • a_{ij} — a with subscript ij
  • \frac{x+1}{x-1} — fraction with expressions
