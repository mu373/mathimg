How to export equations

Export your equations as image files for use in presentations, documents, and websites. MathEdit's SVG exports contain embedded metadata, so you can re-import and edit them anytime.

Re-editable SVG exports

Every SVG exported from MathEdit contains embedded metadata including:

  • The original LaTeX source code
  • The equation label
  • Display mode (block or inline)
  • Generation timestamp
  • Rendering engine version

This means you can:

  • Drag an SVG file back into MathEdit to continue editing
  • Update equations without recreating them from scratch
  • Share SVG files with collaborators who can edit them in MathEdit

Even if you lose the original MathEdit project, you can recover your equations from the exported SVGs.

Export formats

SVG (Scalable Vector Graphics)
Vector format that scales perfectly at any size. Contains embedded LaTeX source for re-editing. Ideal for presentations and documents.
PNG (Portable Network Graphics)
Raster format with transparency. Use when SVG is not supported. Does not contain embedded source.

Export a single equation

  1. Select the equation you want to export.
  2. Choose File > Export > Export SVG... or Export PNG...
  3. Choose a location and click Save.

Keyboard shortcut: ⇧⌘E for SVG

Export all equations

Export your entire project at once:

  1. Choose File > Export > Export All SVGs...
  2. Choose a folder location.
  3. All equations are saved as individual files, named using their labels.

Keyboard shortcut: ⌥⇧⌘E

Copy to clipboard

Copy equations for quick pasting into Keynote, PowerPoint, or other apps:

  • ⇧⌘C — Copy as SVG (re-editable)
  • ⌥⌘C — Copy as PNG
  • ⌥⇧⌘C — Copy LaTeX source

You can also right-click on an equation in the preview pane to access copy options.

Re-import an SVG

To edit a previously exported SVG:

  • Drag the SVG file onto the MathEdit window, or
  • Choose File > Import SVG...

MathEdit extracts the embedded LaTeX source and adds the equation to your current project. The label is preserved, so you can update and re-export with the same filename.
