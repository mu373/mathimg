Getting started with MathEdit

MathEdit helps you manage all the equations for a project in one place. Create a MathEdit document for each presentation or paper, add your equations, then export them as SVG images. Every SVG contains embedded metadata, so you can re-import and edit your equations anytime.

Create a new project

  1. Open MathEdit.
  2. Choose File > New or press âŒ˜N.
  3. A new document opens with one empty equation.

Save your project to keep all equations together. When you need to update an equation later, open the project, make your changes, and re-export.

The MathEdit interface

The main window has three areas:

Sidebar (left)
Shows all equations in your project. Click to select.
Editor (center)
Write your LaTeX code here with syntax highlighting.
Preview (right)
See your rendered equation update as you type.

Write your first equation

  1. Click in the editor area.
  2. Type a LaTeX equation, for example: E = mc^2
  3. The preview updates automatically as you type.

Add more equations

Separate equations with --- on its own line. Use \label{name} to identify each equation:

E = mc^2
\label{energy}

---

F = ma
\label{newton}

Each section becomes a separate equation you can export individually.

Export and re-edit

Export your equations as SVG files. MathEdit embeds the original LaTeX source in each SVG, so you can drag an SVG back into MathEdit to continue editing. No more recreating equations when you need to make changes.
